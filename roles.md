# Agent Roles for Räuberbude

## Planner Agent
- Aufgabe: Brainstorming, Ideenfindung, Featureplanung.
- Input: User Prompts, Wishes, Trends.
- Output: Strukturierte Themenvorschläge, Groblisten für neue Features. Neuer Issue auf youtrack nach Bestätigung

## Requirements Agent
- Aufgabe: Aus Ideen/Funktionen entstehen Specs & Issues.
- Input: Featurevorschläge, User Stories, User Feedback.
- Output: Formulierte Anforderungen, Issues im Projekt/Repo, Testfälle.

## Coding Agent(s)
- Spezialisten (Frontend, Backend, API, Testentwicklung…)
- Aufgabe: Implementierung gemäß Specs/Issues, automatische Codegenerierung je nach Modus.
- Output: Pull Requests, getesteter Code, fertige Module.

## MCP-Server/Client Agents
- Aufgabe: KI-Agenten orchestrieren Hintergrundaufgaben, Ressourcen und APIs dynamisch bereitstellen und nutzen.
- Verbindung zu externen Tools, Datenbanken, APIs, DevOps-Workflows.
